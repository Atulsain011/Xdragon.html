# Xdragon-Gaming-Style-Download-Landing-Page
Xdragon is a modern gaming-style download landing page built using HTML, CSS, and animated UI effects. It features a futuristic design with CRT scanlines, dynamic background animation, start overlay interaction, and a responsive glassmorphism interface.
Xdragon is a visually immersive gaming-themed download page created using pure HTML and CSS.
The project focuses on delivering a futuristic UI experience with smooth animations and game-style design elements.

Features

🎮 Gaming-style futuristic UI

🔥 Animated background with Ken Burns effect

🖥 CRT scanline display effect

🚀 Start overlay screen with interactive button

💎 Glassmorphism card design

📱 Responsive layout

This project is designed as a modern landing page template for game downloads or gaming products.
